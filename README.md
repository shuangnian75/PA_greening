# PA_greening
Code to reproduce the analysis in: Future cropland expansion constrains greening within global protected areas

@SaiQu

Folders are numbered in the order they appear in the Methods section:
protected area filtering (WDPA, dissolve, rasterize)
distance predictors
LUH2-ESA-CCI harmonization and land demand calibration
CMIP6 ensemble processing
kNDVI calculation + grid-specific XGBoost model
counterfactual attribution
trend stats (Sen's slope, Mann-Kendall, GLM, Kruskal-Wallis/Dunn)
